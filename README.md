# Foxora Website

Modern marketing website for Foxora AI — Vite + React + TypeScript + Tailwind v4.

First quick version with the best 4 sections from foxora.ai:

1. **Hero** — "Build your whole startup. In one app."
2. **Meet Your Crew** — 16 specialist agents
3. **Run Your Business** — 6 goal cards
4. **Pricing** — 4 tiers (Free / Starter / Pro / Max)

## Stack

- Vite + React 19 + TypeScript
- Tailwind CSS v4 (via `@tailwindcss/vite` plugin)
- Framer Motion for entrance + hover animations
- Lucide React for icons
- Inter font

## Develop

```bash
npm install
npm run dev      # http://127.0.0.1:5173
npm run build    # production build to dist/
```

## Deploy

Auto-deployed to Vercel on every push to `main`.
